Web Dev Concepts

The purpose of a designated home page is to have one place for the file to return when a blank GET request is received, aka "/", or more broadly, to have an initial or default page users are directed to when they first visist a website.

In the Network tab, you can see the file path and associated requested resources.

The favicon.ico file has a status of 200 because it is a part of the index.html, which as been properly retrieved. The main.css and main.js have a status of 404 because they are not being properly retrieved, hence the "Not Found". This is because main.css and main.js do not exist, and because the client is trying to access them anyways, the status 400 is returned, indicating client error.

The scheme is https, the subdomains are web and engr, the host domain is oregonstate.edu, and the resources are /~secrestc/CS290-ULA/Module-1/a1-secrestc/.
